Test Pagination - Monitor Page

กด F12 เพื่อเปิด Developer Console แล้วคลิกปุ่มด้านล่าง

วิธีทดสอบ:

  1. คลิกปุ่มเพื่อเปิดหน้า Monitor
  2. รอให้โหลดข้อมูลเสร็จ (จะมี pagination ด้านล่าง)
  3. คลิกปุ่ม "2" เพื่อไปหน้า 2
  4. ตรวจสอบว่าข้อมูลเปลี่ยนและหมายเลขหน้าแสดงถูกต้อง
  5. ลองคลิก "ถัดไป →" หรือ "← ก่อนหน้า"
